Title: Flip Square Submatrix Vertically
URL: https://leetcode.com/problems/flip-square-submatrix-vertically/

LLM Explanation:
The problem asks us to vertically flip a square submatrix of size `k x k` within a larger `m x n` grid. The submatrix's top-left corner is at `(x, y)`. A vertical flip is equivalent to reversing the order of the rows within that submatrix.

The most direct way to solve this is using a two-pointer approach, which is a standard technique for reversing sequences.

1.  **Identify Submatrix Rows:** The submatrix occupies rows from index `x` to `x + k - 1`.

2.  **Two-Pointer Setup:** We initialize two pointers:
    *   `top`: points to the first row of the submatrix, starting at `x`.
    *   `bottom`: points to the last row of the submatrix, starting at `x + k - 1`.

3.  **Iterative Swapping:** We loop as long as `top < bottom`. In each iteration:
    *   We swap the row segment of the `top` row with the corresponding segment of the `bottom` row.
    *   The segments to be swapped are defined by the submatrix's columns, which range from `y` to `y + k - 1`.
    *   In Python, this swap can be done concisely using slice assignment: `grid[top][y:y+k], grid[bottom][y:y+k] = grid[bottom][y:y+k], grid[top][y:y+k]`.
    *   After the swap, we move the pointers closer to the center: `top` is incremented, and `bottom` is decremented.

4.  **Termination:** The loop terminates when the pointers meet or cross, at which point all the necessary rows have been swapped, and the submatrix is vertically flipped.

The entire operation is performed in-place on the input `grid`, which is then returned.

**Complexity Analysis:**
*   **Time Complexity:** `O(k^2)`. The `while` loop runs `k / 2` times. Inside the loop, swapping slices of length `k` takes `O(k)` time. Therefore, the total time complexity is `O(k/2 * k) = O(k^2)`.
*   **Space Complexity:** `O(k)`. The in-place modification of the grid means we don't use extra space proportional to the grid size. However, the slice assignment operation creates temporary copies of the row segments being swapped, which requires `O(k)` auxiliary space.
