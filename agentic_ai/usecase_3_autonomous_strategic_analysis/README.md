# Use Case 3: Autonomous Strategic Analysis of Google Cloud

## What this project does

A **genuinely agentic** multi-agent system that answers this executive question:

> *"Analyze Alphabet's latest quarterly earnings, focusing on Google Cloud.
> Identify the biggest strategic threat to Microsoft Azure.
> Generate three actionable mitigation strategies."*

Every reasoning step is **LLM-powered** (Gemini 2.5 Flash).
Every piece of data comes from **real internet sources** — no mock data.

## What makes this genuinely agentic (not a mock-up)

| Capability | How it works |
|---|---|
| **LLM Planning** | AnalystAgent asks Gemini to decompose the prompt into execution steps |
| **Real Data** | SEC EDGAR XBRL, Google News RSS, Gemini Google-Search grounding |
| **Hybrid Retrieval** | BM25 (sparse) + Gemini embeddings (dense) → RRF fusion |
| **LLM Reranking** | Gemini scores chunk relevance 0-1, drops noise |
| **LLM Extraction** | Gemini extracts structured financials from raw web text |
| **Dynamic Tool** | Market-share-delta function generated & loaded at runtime |
| **LLM Critique** | Rubric-scored self-critique (Depth / Evidence / Causality / Actionability) |
| **Refinement Loop** | If critique says NEEDS_REFINEMENT → LLM generates new queries → re-retrieves → re-analyses |
| **LLM Memory** | Gemini extracts relationship triples written to GraphMemory at runtime |
| **LLM Synthesis** | Executive summary + strategy table generated by Gemini |
| **Cost Guard** | Hard budget cap, per-call logging, cache-hit tracking, dry-run mode |

## Billing protection (important for learning)

```
                      ┌─────────────────────────────────────┐
                      │  Gemini 2.5 Flash  =  CHEAPEST      │
                      │  $0.15 / 1M input   $0.60 / 1M out  │
                      │  Google AI Studio free tier:          │
                      │    15 RPM · 1M TPM · 1500 RPD · FREE │
                      │  Embeddings: text-embedding-004 FREE  │
                      │  Grounding: 500 free queries/day      │
                      └─────────────────────────────────────┘
```

Safety measures built in:
1. **Hard budget cap**: default $0.50 — execution halts if exceeded
2. **Aggressive caching**: every response cached on disk — re-runs cost $0
3. **Dry-run mode**: `--dry-run` flag skips ALL API calls
4. **Token limits**: reduced across the board (4096 max output, tight context budgets)
5. **Lean retrieval**: BM25 weighted higher (free) vs. embeddings; fewer rerank chunks
6. **Cost summary**: printed after every run with exact token counts and USD estimate

Expected cost per run: **$0.01 – $0.05** (flash pricing). Re-runs with cache: **$0.00**.

## Quick start

```bash
# 1. Set your API key
echo "GEMINI_API_KEY=your-key-here" > .env

# 2. Install deps (already in project venv)
pip install -r requirements.txt

# 3. Run (dry-run first to verify plumbing)
python main.py --dry-run

# 4. Real run (uses Gemini API)
python main.py

# 5. Custom budget
python main.py --budget 0.25

# 6. Clear cache and re-run fresh
python main.py --clear-cache
```
